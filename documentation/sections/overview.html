2. Evaluation Result ontology Overview back to ToC

The Evaluation Result ontology (EVAL) is an ontology for describing evaluation results. It enables capturing knowledge of values obtained in an evaluation of products, services, or actions. The following table lists the classes, properties and data properties that are used in this ontology.

  • Evaluation
  • EvaluationData
  • EvaluationSubject
  • QualityValue
  • SubjectCategory
  • Instant
  • QualityMeasure
  • Unit_of_measure
  • Measurement_scale

  • obtainedFrom
